# dotfiles

Cross-platform dotfiles repo for macOS and Linux (Raspberry Pi).

## Repo structure

```
dotfiles/
├── shared/shell/         # Cross-platform zsh configs
│   ├── .zshenv           → ~/.zshenv (ZDOTDIR + XDG)
│   ├── .zshrc            → ~/.config/zsh/.zshrc
│   ├── aliases.zsh       → ~/.config/zsh/aliases.zsh
│   └── .p10k.zsh         → ~/.config/zsh/.p10k.zsh
├── shared/git/
│   ├── .gitconfig        → ~/.gitconfig (uses [include] for platform overrides)
│   └── ignore            → ~/.config/git/ignore
├── macos/shell/.zprofile → ~/.zprofile (Homebrew + nvm)
├── macos/hammerspoon/    → ~/.hammerspoon/ (window management)
├── macos/launchd/        → ~/Library/LaunchAgents/ (COPIES, not symlinks)
├── macos/scripts/        → ~/.local/bin/* (automation scripts)
├── macos/defaults.sh     # macOS system preferences (opt-in)
├── linux/shell/.zprofile → ~/.zprofile (nvm only)
├── bin/                  → ~/.local/bin/* (cross-platform utilities)
├── bootstrap.sh          # OS-detecting installer
├── Brewfile              # Homebrew packages (macOS)
└── dotfiles.env.example  # Template for private config
```

## Conventions

- Symlinks for configs and scripts, copies for LaunchAgent plists
- Scripts deploy to `~/.local/bin/` (XDG-compliant, on PATH)
- Platform branching: `if [[ "$OSTYPE" == darwin* ]]`
- Private values (NTFY_URL, PI_HOST) sourced from `~/.config/dotfiles/env`
- Git platform overrides via `[include] path = ~/.config/git/local.gitconfig`
- 2-space indentation, `set -euo pipefail` for bash scripts
- Zsh scripts use `#!/bin/zsh`, bash scripts use `#!/bin/bash`

## Deployment

Run `./bootstrap.sh` — it detects macOS vs Linux and deploys accordingly.
Idempotent: backs up existing files, overwrites symlinks.

## LaunchAgents

All plists reference `__HOME__/.local/bin/<script>`.
Plists are COPIED (not symlinked) because launchd removes symlinks on unload.

| Agent | Schedule | Script |
|---|---|---|
| backup-claude | WatchPaths | /bin/cp (no script) |
| mac-backup | 03:00 daily | mac-backup.sh |
| mac-health-check | 08:00 daily | mac-health-check.sh |
| mac-stats-push | Every 5 min | mac-stats-push.sh |
| brew-maintenance | Sun 09:00 | brew-maintenance.sh |
| obsidian-weekly-note | Mon 00:05 | obsidian-weekly-note.py |

## Key paths

- Repo: `~/repositories/dotfiles/`
- Private config: `~/.config/dotfiles/env` (gitignored via dotfiles.env)
- Platform git overrides: `~/.config/git/local.gitconfig` (generated by bootstrap)
- Scripts: `~/.local/bin/` (symlinks to repo)
- ZDOTDIR: `~/.config/zsh/`
